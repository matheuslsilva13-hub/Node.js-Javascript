## 💻 Configuração de Ambiente e Estrutura Sequencial com Node.js
**Este repositório contém a resolução de exercícios de lógica de programação utilizando Node.js, realizando a transição da sintaxe da linguagem C para JavaScript.**

## Instalação do Virtual Studio e Node.js Pelo terminal
**Instalação Nide.js:** sudo apt update | sudo apt install nodejs npm -y

**Instalação Virtual Studio Code:** sudo snap install --classic code

## 🚀 Execução
**Para executar os scripts individualmente, utilize o comando no terminal :**
**node ex01.js**

## 📂 Estrutura do Projeto
 **01_EstruturaSequencial: Cálculos matemáticos e operadores básicos.**
 
 **02_EstruturaDeDecisao: Condicionais e operadores de igualdade estrita (===).**
 
 **03_EstruturaDeRepeticao: Laços de iteração (for, while, do...while).**
